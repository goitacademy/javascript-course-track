## 1. Пограємо в Kahoot і обговоримо питання
## 2. Розберемо роботу з колекцією даних
- Робота з колекцією даних
- Сервіс [mockapi](https://mockapi.io/)
- Робота з приватним API
  - **C**reate
    - Метод POST
    - Тіло
    - Заголовки
    - Відповідь
  - **R**ead
    - Метод GET
    - Відповідь
  - **U**pdate
    - Методи PUT та PATCH
    - Тіло
    - Заголовки
    - Відповідь
  - **D**elete
    - Метод DELETE
    - Відповідь
